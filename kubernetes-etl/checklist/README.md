# Чек-лист проверки развертывания

- [ ] Установлен инструмент пакетного менеджера `winget`.
- [ ] Выполнена команда `winget install -e --id derailed.k9s`.
- [ ] Выполнена команда `winget install k3s` и кластер успешно стартовал.
- [ ] Контекст `kubectl` указывает на правильный кластер (локальный k3s или Gcloud).
- [ ] Развернут Helm chart командой `helm install knowledge-graph .` из папки `additional`.
- [ ] Под СУБД PostgreSQL запущен (`Running`) в `k9s`.
- [ ] Под с задачей DBT (Job) успешно отработал (`Completed`) и выполнил трансформации.
- [ ] В СУБД PostgreSQL существуют таблицы `nodes` и `edges`.
- [ ] Запрос к `kg_search_view` успешно возвращает связи (Сплав X -> Режим Y -> Эффект Z).
